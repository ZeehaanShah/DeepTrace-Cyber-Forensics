# DeepTrace Cyber Forensics

AI-powered phishing detection and cyber forensics platform that analyzes URLs, webpage content, and AI-generated threats using machine learning and browser extension integration.

## Features
- URL-based phishing detection
- Text/content phishing analysis
- AI-generated threat detection
- Browser extension integration
- FastAPI backend
- Ensemble ML models

## Tech Stack
- Python
- FastAPI
- XGBoost
- LightGBM
- DeBERTa
- JavaScript
- Chrome Extension APIs
